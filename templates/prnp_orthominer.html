Análisis BLAST · Identificación de ortólogos de PRNP · Genomas de mamíferos

Acerca de PRNP-OrthoMiner
Pipeline computacional diseñado para identificar y extraer sistemáticamente las secuencias de la pauta de lectura abierta (ORF) de PRNP a partir de ensamblajes de genoma completo de mamíferos. Emplea una estrategia de alineamiento local tipo BLAST frente a una secuencia de referencia de PrP para identificar la región genómica con mayor similitud, extraer el contexto de secuencia circundante y generar archivos FASTA con los loci candidatos de PRNP.

Flujo del pipeline

1
Lectura del genoma
Localiza el ensamblaje genómico en la ruta especificada (formato FASTA/FA/FNA).
2
Construcción de base de datos BLAST
Crea una base de datos BLAST local a partir del ensamblaje de la especie objetivo.
3
Alineamiento BLAST
Ejecuta TBLASTN usando la secuencia de referencia PrP seleccionada para identificar la región genómica con mayor similitud.
4
Extracción del locus candidato
Extrae la región genómica con contexto flanqueante para capturar el ORF completo.
5
Generación de FASTA
Produce archivos FASTA con los loci candidatos de PRNP listos para anotación o análisis filogenético.

Formatos admitidos

  • FASTA Ensamblajes de genoma completo
  • FA / FNA Variantes de formato FASTA
  • GZ Archivos comprimidos (descompresión automática)
Los genomas pueden descargarse de NCBI Assembly o Ensembl.
